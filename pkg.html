Using PKGNG with FreeBSD

Nov 01 2013 By Abhay

pkgng is new version of package manager to install, upgrade and remove packages from FreeBSD system.

Requirements

  • FreeBSD 9.1 or newer

Installation

  • pkgng must be install from ports to get latest version
    • On terminal:
    • cd /usr/ports/ports-mgmt/pkg
    • make
    • make install clean

Note: One pkgng is installed one must convert existing pkg database to newwer format. Note: In case of Major conflict for package just reinstall that particular package from pkgng

pkg2ng It will automatically take care of conflicts.

Important: This will be irreversible step, Once done you can't use Older package management tools.

  • To make sure that your ports tree will also use same pkgng tool, Update your make.conf as follows: WITH_PKGNG = yes

Usage

  • To get help pkg help install
  • Info for particular package pkg info pkg
  • Installing particular variant of the software
