PwlessSSH Deployment

Oct 03 2011 By Abhay

Enable login without password ssh

- Tired of typing SSH password at prompt

- During runtime in automated scripts?

Requirements:

  • Interconnectivity between both servers that you enable password less login.
  • SSH server should be running on server that you want to enable pwless login.
  • If you have firewall enabled then open port 22 in it.

Setup

  • Just type in ssh USERNAME@IPADDRESS to make sure you can login to the server and exit.
  • At the prompt, again type

    ssh-copy-id USERNAME@IPADDRESS

  • it will create authorizedkey file in the .ssh directory on the target system.
  • It will place authorized_key file in you .ssh directory.
  • After doing that you will be able to login to remote server without need for password.

Verification:

  • Now your login to the remote box should be pwless.
  • If any problems please check your authorized_keys file.

SSH Forever
